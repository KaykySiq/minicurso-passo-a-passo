Sobre o Projeto

Aprenda back-end através de um produto real.

Teoria é ótima mas nada ensina como colocar algo real no ar. Este projeto te guia pela construção do núcleo do Twitter usando Django e Bootstrap 5: o laboratório perfeito para entender operações CRUD, autenticação, bancos de dados relacionais e feeds dinâmicos.

Ao final, você terá uma aplicação pronta para o portfólio e a confiança para criar suas próprias ideias full-stack.

Python & Django Bootstrap 5 SQLite / Postgres Auth & Sessões Arquitetura MVT Git & GitHub
O Que Você Vai Aprender

Um currículo focado, do início ao fim.

Do primeiro model a um feed autenticado e funcional cada etapa é pensada para construir habilidades práticas e transferíveis.

Backend com Django

Models, views, URLs e a robusta arquitetura MVT que sustenta apps em produção.

Interface com Bootstrap 5

Criando interfaces responsivas e modernas sem se preocupar com CSS do zero.

Banco de Dados e CRUD

Gerenciamento de posts, usuários e datas com persistência confiável via ORM.

Autenticação

Login, logout e gerenciamento de sessões de usuário do jeito certo.
